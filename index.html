📦 Sistema de Etiquetas de Notas Fiscais

Projeto desenvolvido com Django, SQLite e Bootstrap para geração de etiquetas com QR Code da chave NFe.

🔐 Acessar Login

🔧 Funcionalidades

  • Cadastro e login de usuários com autenticação
  • Dashboard com filtros por nome, controle e data
  • Visualização de etiquetas com dados da nota fiscal
  • Geração automática de QR Code da chave NFe
  • Painel administrativo para gerenciar usuários

🌐 Deploy

Este projeto pode ser publicado em plataformas como Render, Railway ou Heroku. Veja instruções no README.
